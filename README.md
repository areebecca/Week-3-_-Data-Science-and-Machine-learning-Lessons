## Python Programming Fundamentals Lab
# Objective
To master the core building blocks of Python programming, focusing on dynamic data handling, logical flow control, and efficient code structuring using built-in functions and libraries.
# Tools Used

Language: Python 3.x

Environment: Google Colab 

Libraries: Pandas (for data manipulation) and functools (for data reduction).

# Step-by-Step Process

String Formatting: Implemented multiple techniques (F-strings, .format(), and str()) to create readable, variable-driven outputs.

Control Flow Logic: Developed decision-making structures using if-elif-else statements and concise ternary operators.

Iterative Processing: Utilized for loops to iterate through lists and dictionaries, and while loops for condition-based repetition.

Functional Programming: Created reusable code blocks with standard def functions and inline lambda expressions.

Data Transformation: Applied high-order built-in functions like map(), filter(), and reduce() to process and aggregate numerical data.

Structural Data Handling: Explored the basics of Pandas for loading datasets and combining DataFrames using concatenation.

# Key Observations / Lessons Learned

Readable Code: F-strings provide the most efficient and readable way to handle string interpolation in modern Python.

Optimization: Using map() and filter() can often replace multi-line loops, making data processing scripts more concise.

Reusability: Transitioning from simple scripts to modular functions is essential for building scalable data analysis tools.

Data Structure Utility: Understanding the axis parameter in Pandas is critical for correctly merging or dropping data during the cleaning phase.
